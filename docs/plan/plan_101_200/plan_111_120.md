# Plan Phase 111-120 Summary

This file is a compact planning index for agents. Keep `docs/plan/plan.md` as the source of truth.

## phase_111 - Evidence Normalization Boundary

Phase Goal
- Make source eligibility, prompt escaping, source audit, and fallback evidence generation available through an explicit backend boundary.

Completed Work
- Added a focused evidence normalization module for source document IDs, `as_of_at` decisions, source audit, untrusted prompt context, prompt-budget exclusion, and deterministic fallback evidence items.

Next Steps
- Continue to use this boundary before prompt construction and scoring.

## phase_112 - Prediction Orchestration Routing Guard

Phase Goal
- Keep news digest, market snapshot, prediction, and PnL routing from stepping on each other.

Completed Work
- Fixed the exact-ticker news path so `AAPL latest news` routes to news digest instead of market snapshot.

Next Steps
- Continue expanding intent matrix tests before changing route precedence.

## phase_113 - Confidence Factor Scoring Metadata

Phase Goal
- Make confidence behavior inspectable instead of only returning a numeric score.

Completed Work
- Added scoring confidence factors for eligible evidence, thin evidence, stance diversity, and excluded-source penalty.

Next Steps
- Keep confidence factors stable as a user-facing explanation boundary.

## phase_114 - Analysis UI Operational State

Phase Goal
- Surface provider/model and confidence-factor state near the analysis result.

Completed Work
- Added frontend rendering for provider, model, and confidence factors in the analysis panel.

Next Steps
- Keep operational state close to source audit and probability output.

## phase_115 - News Digest Transparency UX

Phase Goal

Next Steps

## phase_116 - Prediction Cache Schema Boundary

Phase Goal

Completed Work

## phase_117 - PnL Evaluation Artifact Separation

Phase Goal

Next Steps

Phase Goal

Completed Work

Next Steps
